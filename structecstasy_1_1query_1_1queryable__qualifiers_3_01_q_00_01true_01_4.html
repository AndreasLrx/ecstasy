Detailed Description

template<thread::Lockable Q>
struct ecstasy::query::queryable_qualifiers< Q, true >

Get the queryable type with the correct qualifiers.

By default this is a reference to the type Q. If Q is thread::Lockable and AutoLock is true, then the type is a thread_safe_queryable_t<Q>. It is not a reference to the type because it will be implicitly constructed using the thread::LockableView<Q> constructor (taking Q&). Meaning a ThreadSafeQuery will have LockableView<Q> in its storages but Q& in its constructor.
