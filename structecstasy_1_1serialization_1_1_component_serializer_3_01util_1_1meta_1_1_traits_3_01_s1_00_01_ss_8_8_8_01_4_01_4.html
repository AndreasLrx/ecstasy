Detailed Description

template<typename S1, typename... Ss>
struct ecstasy::serialization::ComponentSerializer< util::meta::Traits< S1, Ss... > >

Serialize a component with the given serializer.

Serializer is passed as an ISerializer but with its type_info.

Note
This struct is used to counter the limit of the C++ language that doesn't allow to have a virtual template.
Template Parameters
SRegistered serializer types wrapped inside an util::meta::Traits.
