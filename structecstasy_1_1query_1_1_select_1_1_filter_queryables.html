inlinestaticconstexpr

Primary template recursively called until there is no nextPivot.

It calls itself with:

  • ContainsPivot to to true if nextPivot is in the SelectedQueryables types.
  • lefts to the current lefts with pivot (only if ContainsPivot is currently true).
  • pivot to current nextPivot.
  • nextPivot and rights are set or not depending on the queryable lefts in the current rights. If rights is empty, it will call the value() without nextPivot value to finalize the result.

[in]rightsRemaining queryables, first one will become the nextPivot of the recursive call.
