Detailed Description

Listener of a button sequence.

Button sequences of buttons A, B, X are triggered only when A, B and X are pressed and released one at a time (A, then B, then X). If any other button is pressed while waiting for a button release or if the button was not expected, the validated buttons are reset: A, B, Y, X will not work because the Y reset the sequence.
