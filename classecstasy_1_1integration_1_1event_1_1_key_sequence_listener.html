Detailed Description

Listener of a key sequence.

Key sequences of keys A, B, C are triggered only when A, B and C are pressed and released one at a time (A, then B, then C). If any other key is pressed while waiting for a key release or if the key was not expected, the validated keys are reset: A, B, Z, E will not work because the Z reset the sequence.
