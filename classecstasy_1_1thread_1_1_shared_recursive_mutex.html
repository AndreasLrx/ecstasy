Detailed Description

Wrapper for std::shared_mutex allowing recursive locking by the same thread.

Note
Recursive locking is only allowed if the thread locks the mutex with exclusive access.
Warning
Deadlock always appears if a thread locks the mutex as shared and then tries to lock it normally.
