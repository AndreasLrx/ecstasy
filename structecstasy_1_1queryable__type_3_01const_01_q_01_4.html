Detailed Description

template<query::Queryable Q>
requires query::ConstQueryableObject<Q>
struct ecstasy::queryable_type< const Q >

Try to get the type of a queryable associated to a type stored in the registry (component, storage, resource...).

Returns C if:

  • C is derived from the IResource class and match the Queryable concept.
  • C match the IsStorage concept and match the Queryable concept.
  • C is a const type matching the ConstQueryableObject concept.
  • C match the ecstasy::query::Modifier Returns C::Modifier if C match the RegistryModifier concept. Otherwise, returns getStorageType<C>.
