inlinestatic

Execute a AND query with all the given Queryables but returns only the one in Selected.

Warning
All queryables specified in Selected must be passed as parameters in the same order (independing from the one non selected). For example: Selected<A, B, C> must have at least an instance of type A, B and C in the parameters in the same order: where<A, U, V, B, C, X, Z> (which leads to where<A, B, C> if we omit non selected queryables)
